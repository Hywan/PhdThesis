My PhdThesis. This is a “subtree” of a private SVN repository.

# Compile

How to compile the document? Blazing easy:

    $ cd PhdThesis
    $ make bib
    $ make
    $ make open # on Darwin

A PDF is produced in `PhdThesis/Dist/`.
